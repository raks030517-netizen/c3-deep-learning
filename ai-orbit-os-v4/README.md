# AI ORBIT OS v4

이 버전은 빈 프로젝트에서 바로 실행 가능한 Next.js 풀프로젝트입니다.

## 실행

```bash
npm install
npm run dev
```

접속:

```text
http://localhost:3000
```

## 페이지 구조

- `/` : 운영체제형 홈
- `/galaxy` : AI 개념 은하계 지도
- `/academy` : 강의자료 기반 커리큘럼 로드맵
- `/mission/[slug]` : 개념별 전용 실험실

## 포함 미션

ML 개요, Train/Dev/Test, Scaling/Leakage, k-NN, Regression, Gradient Descent,
Ridge/Lasso, Classification, Metrics, Bias/Variance, Decision Tree, Random Forest,
K-Means, PCA, Perceptron/XOR, Neural Network, Backpropagation, Overfitting/Dropout,
CNN, Transfer Learning/MobileNet, Object Detection, RAG/LLM.
